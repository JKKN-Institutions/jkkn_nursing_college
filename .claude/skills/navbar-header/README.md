# Navbar/Header Component Skill

A fully responsive, multi-level navigation component for Next.js applications with Tailwind CSS.

## Features

- **Responsive Design**: Mobile hamburger menu and desktop horizontal navigation
- **Multi-level Menus**: Support for nested submenus (up to 3 levels deep)
- **Hover & Click Interactions**: Desktop hover menus, mobile click-to-expand
- **Body Scroll Lock**: Prevents scrolling when mobile menu is open
- **Customizable Styling**: Uses Tailwind CSS for easy customization
- **TypeScript Support**: Full type safety with TypeScript interfaces
- **Search Functionality**: Built-in search icon (functionality to be implemented)
- **Fixed Header**: Stays at the top while scrolling

## Quick Start

1. Copy `Header.tsx` to your `components/` directory
2. Copy `siteData.ts` to your `data/` directory (or merge with existing)
3. Update the logo path in `Header.tsx` (line 46)
4. Customize navigation menu in `siteData.ts`
5. Add `primary` color to your Tailwind config
6. Import and use in your layout
